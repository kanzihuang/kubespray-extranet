# 公网创建 kubernetes 集群

## 背景

近年来各大云厂商为了拓展新用户，为新用户提供了力度很大的优惠活动，个人用户购买云主机组建 Kubernetes 集群的费用大幅下降。由于云主机上仅配置了内网地址，内网地址不能实现相互访问，组建集群存在很大不便，下面依次介绍组建公网集群的问题和解决方案。
